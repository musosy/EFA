# Express Fast API

## About

This is an Express template built with:
* Typescript
* Jest
* Prisma
* Docker
* Eslint
* Prettier
* GitHub Actions

## Commands
To build the app:
```bash
npm run build
```

To run the dev app:
```bash
npm run start:dev
```

To run the prod app:
```bash
npm run start:prod
```

To run tests:
```bash
npm run test
```

To run tests with coverage:
```bash
npm run test:cov
```

To run the linter:
```bash
npm run lint
```
